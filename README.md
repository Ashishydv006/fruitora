# Fruitora - React Frontend (Vite + Tailwind)

## Setup
1. Ensure Node.js (v16+) and npm are installed.
2. Run `npm install` to install dependencies.
3. Run `npm run start` to start dev server (Vite default port 3000).

## Build
- `npm run build` then `npm run preview` to preview production build.

This project uses Vite + React + Tailwind CSS. Replace placeholder content and images as needed.
